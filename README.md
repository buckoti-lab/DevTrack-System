<h1 align="center">Software Development Management System</h1>

<p align="center">
A Laravel-based platform for managing software project requests, proposals, SRS documents, and development workflows.
</p>

---

## About The Project

This system allows clients and developers to collaborate efficiently by providing a structured way to:

* Submit project proposals, SRS, and quotations
* Track project progress in real-time
* Manage tasks and developer workload
* Maintain transparency throughout the software development lifecycle

---

## Features

* Proposal & SRS submission system
* Real-time dashboard (AJAX powered)
* Role-based access (Admin, Developer, Client)
* Document preview (PDF, images)
* Task tracking & progress monitoring
* Secure authentication system

---

## Built With

* Laravel (PHP Framework)
* MySQL
* HTML, CSS, JavaScript (AJAX, jQuery)
* Bootstrap

---

## Installation

```bash
git clone https://github.com/buckoti-lab/DevTrack-System.git
cd DevTrack-System
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
